# Project RECALL

Project Recall is an early stage Alzhimers detection device. It features an E-ink display, microphone, and hardware that allows users to know if they are at risk for Alzhimers.

Why Build This:

Millions suffer from Alzhimers, but if detected and treated early enough, the disease is preventable. My friend Subha, is working on research that uses machine learning and an extensive database to detect Alzhiemers in patients with high a high rate of accuracy. However, to do so he needs to have a way to monitor a patient's voice throughout the day so he can use his algo to see if the patient is at risk for Alzhimers. Unfortunately, Subha isn't too well versed in hardware design, so I'm hoping to design a solution to his problem.

Criteria:

There's a couple things I need from the project + some other features that would be awesome to have. The nessecary features are as follows

- A microphone that's capable of accurtely picking up a patient's voice
- A E-ink dispaly that display connection status and alerts to the patient
- Hardware that sends data to a iPhone in heartbeats to process
- Long battery life & rechargable

There's also some features that aren't required but would be awesome to have. These are a few

- A companion app that delivers more in depth analytics based off whats collected from the hardware.
- A speaker that doesn't abuse the battery would be awesome
- A cool looking enclosure for the hardware
